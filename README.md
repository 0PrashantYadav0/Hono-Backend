# Hono Backend
## How to get started : 
### To install dependencies:
```sh
bun install
```

### Add .env file as per given .env.example file
```env
MONGO URI=<your mongodb url>
```

### To run:
```sh
bun run dev
```

Open
```
http://localhost:3000
```

## To run Index.js
```json
"scripts": {
  "dev": "bun run --hot src/app.ts"
  },
```
